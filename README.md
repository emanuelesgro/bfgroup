# Sostenibilità Gruppo BF S.p.A. – Progetto universitario

Questo repository raccoglie un progetto universitario in cui ho creato una pagina web per presentare in modo più immediato il **Piano Strategico 2023–2027** del Gruppo BF S.p.A. L’obiettivo è rendere i contenuti principali più facili da consultare rispetto al PDF completo: prima una panoramica chiara sul sito, poi  la possibilità di approfondire con il documento ufficiale.

Dentro la pagina ho organizzato le informazioni in sezioni, così si naviga velocemente tra i temi: introduzione, i pilastri strategici legati alla sostenibilità: ambiente, biodiversità, tracciabilità e responsabilità sociale e una parte con risultati e certificazioni, oltre al link per scaricare la Dichiarazione Non Finanziaria.

## Tecnologie
Il sito è fatto con HTML, CSS e JavaScript vanilla, senza framework.

## Come usarlo
- In locale: scarica la cartella del progetto e apri `Untitled-1.html` con un browser.
- Online: la pagina è pubblicata con GitHub Pages.
